# SproutRob
## THIS IS THE __ONLY OFFICIAL__ AND __REAL REPO__ AND ONLY AVAILABLE HERE !!!
This in a custom software for the ESP32 wroom dev board only, links are below.

## __Required conponents__ :
### - 0,96 Inches, 128x64 I2C
### - Some wires
### - 1 Big breadboard or 2 littles
### - A capacitor (Optional)


## __Hardware :__ 
1. SCL needs to be bound to the 21 port on the ESP32
2. SDA needs to be bound to the 22 port on the ESP32
3. VCC connected to the 3.3v port on the ESP32
### (You can put a capacitor for a better security between the GND and 3.3v connection ON THE SCREEN NOT THE ESP32 !!!!!!!)
